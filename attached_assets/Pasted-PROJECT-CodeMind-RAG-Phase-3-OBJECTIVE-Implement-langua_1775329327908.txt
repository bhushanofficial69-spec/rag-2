2. **Implement Line Number Tracking**
   Challenge: RecursiveCharacterTextSplitter doesn't track line numbers.
   Solution:
   - Split code into chunks
   - For each chunk, find its position in original code
   - Count newlines before chunk to get start_line
   - Count newlines in chunk to get end_line
   - Store in CodeChunk metadata
   
   Example:
```python
   def get_line_numbers(original_code: str, chunk_content: str, chunk_index: int) -> Tuple[int, int]:
       lines_before = original_code[:original_code.find(chunk_content)].count('\n')
       lines_in_chunk = chunk_content.count('\n')
       return (lines_before + 1, lines_before + lines_in_chunk + 1)
```

3. **Implement Function Name Extraction**
   For each chunk, extract function/class name:
   - Python: Use ast.parse() to find all functions/classes, match to chunk
   - JavaScript/TypeScript: Use regex: /(?:function|const|class)\s+(\w+)/
   - Java: Use regex: /(?:public|private|protected)?\s*(?:static)?\s*(?:class|interface)\s+(\w+)/
   - Store in CodeChunk.function_name (None if not found)
