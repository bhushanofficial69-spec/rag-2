5. **Create services/embedding_cache.py**
   Implement EmbeddingCache class (optional but recommended):
```python
   class EmbeddingCache:
       def __init__(self, max_size: int = 10000):
           self.cache = {}  # {text_hash: vector}
           self.hits = 0
           self.misses = 0
       
       def get(self, text_hash: str) -> Optional[List[float]]:
           if text_hash in self.cache:
               self.hits += 1
               return self.cache[text_hash]
           self.misses += 1
           return None
       
       def set(self, text_hash: str, vector: List[float]):
           self.cache[text_hash] = vector
       
       def stats(self) -> dict:
           total = self.hits + self.misses
           hit_rate = self.hits / total if total > 0 else 0
           return {
               "hits": self.hits,
               "misses": self.misses,
               "hit_rate": hit_rate,
               "size": len(self.cache)
           }
```
