4. **models/schemas.py - Pydantic Models**
   Create request/response models:
```python
   class IngestRequest(BaseModel):
       repo_url: str  # https://github.com/owner/repo
       branch: str = "main"
   
   class IngestResponse(BaseModel):
       status: str  # "processing" | "queued"
       job_id: str
       message: str
   
   class IngestionStatus(BaseModel):
       job_id: str
       status: str  # "processing" | "completed" | "failed"
       files_indexed: int
       chunks_created: int
       error: Optional[str]
       progress_percent: int
   
   class HealthResponse(BaseModel):
       status: str  # "ok"
       timestamp: str
       services: Dict[str, str]  # qdrant, groq, huggingface status
```

5. **services/repo_cloner.py - Repository Cloning**
   Implement RepoCloner class with:
   - Method: clone_repo(repo_url: str, branch: str, temp_dir: str) -> str
   - Returns: path to cloned directory
   - Features:
     * Validate GitHub URL format
     * Use GitPython to clone
     * Handle authentication with GITHUB_TOKEN
     * Timeout: 5 minutes per clone
     * Error handling: invalid URL, network error, permission denied
   - Logging: DEBUG before clone, INFO after success, ERROR if failed

6. **services/file_filter.py - File Filtering**
   Implement FileFilter class with:
   - Method: get_code_files(repo_path: str) -> List[str]
   - Rules:
     * Include: .py, .js, .jsx, .ts, .tsx, .java
     * Exclude directories: node_modules, .git, __pycache__, dist, build, venv, .venv, vendor, .next, .nuxt
     * Skip files > 100KB
     * Returns: list of file paths (absolute paths)
   - Logging: INFO for each filtered directory

7. **services/ingestion.py - Main Ingestion Orchestrator**
   Implement IngestionService class with:
   - Method: ingest_repo(repo_url: str, branch: str) -> job_id
   - Flow:
     1. Validate input (URL format, branch name)
     2. Clone repo to temp directory
     3. Get all code files (use FileFilter)
     4. Log stats (X files found)
     5. For each file:
        a. Read file content
        b. Detect language
        c. (Phase 3) Will chunk here
     6. Return job_id
   - Job tracking: Use in-memory dict {job_id: IngestionStatus}
   - Error handling: Cleanup temp files on failure

8. **routers/health.py - Health Check Endpoint**
   Implement GET /api/health:
   - Response: HealthResponse
   - Check status of: Qdrant (offline for now), Groq (offline), HuggingFace (offline)
   - Status values: "connected" | "disconnected" | "unknown"
   - Include: timestamp, uptime_seconds

9. **routers/ingest.py - Ingestion Endpoints**
   Implement two endpoints:
   
   a) POST /api/ingest
      - Request: IngestRequest (repo_url, branch)
      - Response: IngestResponse (status, job_id)
      - Status code: 202 (Accepted - async)
      - Flow:
        1. Validate request
        2. Create job_id (use uuid4)
        3. Start background task (BackgroundTasks)
        4. Return immediately with job_id
   
   b) GET /api/ingest/status/{job_id}
      - Response: IngestionStatus
      - Status codes:
        * 200: Found (completed, processing, or failed)
        * 404: Job ID not found
      - Return: current status with progress_percent, files_indexed, error if any

10. **main.py - FastAPI App Entry Point**
```python
    from fastapi import FastAPI
    from fastapi.middleware.cors import CORSMiddleware
    
    app = FastAPI(title="CodeMind RAG API", version="0.1.0")
    
    # CORS: Allow frontend localhost:3000
    app.add_middleware(
        CORSMiddleware,
        allow_origins=["http://localhost:3000", "http://127.0.0.1:3000"],
        allow_credentials=True,
        allow_methods=["*"],
        allow_headers=["*"],
    )
    
    # Include routers
    app.include_router(health.router)
    app.include_router(ingest.router)
    
    @app.on_event("startup")
    async def startup():
        logger.info("CodeMind RAG API starting...")
    
    if __name__ == "__main__":
        uvicorn.run(app, host="0.0.0.0", port=8000)
```
